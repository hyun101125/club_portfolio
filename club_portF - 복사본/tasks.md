> 이 파일은 완료한 작업(프로젝트)를 정리하는 용도입니다.  
> 각 항목에는 **상세 설명, 사용 기술, 데모/코드 링크**를 포함하세요.

## 1. 개인 포트폴리오 웹 페이지

- **요약**
  - 반응형 UI를 갖춘 개인 포트폴리오 웹 페이지로, 프로젝트와 기술 스택, 자기소개, 연락처 정보를 한 곳에서 보여줍니다.
- **상세 설명**
  - HTML, CSS, JavaScript로 구축된 싱글 페이지 형태의 포트폴리오입니다.
  - 상단 고정 네비게이션과 히어로 섹션, About, Projects, Skills, Contact 섹션으로 구성되어 있습니다.
  - Firebase Firestore를 백엔드로 사용하여 프로젝트와 스킬 데이터를 동적으로 불러오도록 설계되었습니다.
  - 모바일, 태블릿, 데스크톱 해상도에 맞춰 레이아웃이 자연스럽게 재배치되도록 반응형 스타일을 적용했습니다.
- **사용 기술**
  - Frontend: HTML, CSS (Custom, Flex/Grid), JavaScript (ES Modules)
  - Backend / BaaS: Firebase (Firestore)
  - 기타: 반응형 웹 디자인, 서버리스 아키텍처
- **데모 링크**
  - Live Demo: _예시_) `https://your-portfolio-demo.example.com`
  - Code Repository: _예시_) `https://github.com/your-id/portfolio`
